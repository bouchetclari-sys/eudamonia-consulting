# EudamonIA Consulting Landing Page

Landing page premium, responsive e interactiva para una consultora SaaS/enterprise de IA aplicada al retail tecnológico.

## Archivos
- `index.html`
- `styles.css`
- `script.js`

## Cómo publicarla y obtener URL pública
1. Entrá a https://vercel.com/new
2. Subí esta carpeta o conectala a GitHub.
3. Framework preset: Other.
4. Build command: vacío.
5. Output directory: vacío o raíz.
6. Deploy.
7. Vercel te dará una URL tipo `https://eudamonia-consulting.vercel.app`.

También podés subirla a Netlify arrastrando la carpeta completa en https://app.netlify.com/drop.
